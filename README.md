# Goggins of Monkstown

A modern website for Goggins of Monkstown, a historic Dublin pub founded in 1935.

## Tech Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser.

## Project Structure

- `/app` - Next.js App Router pages
- `/components` - Reusable React components
- `/data` - JSON data files (hours, menu, products)
- `/lib` - Utility functions and contexts
