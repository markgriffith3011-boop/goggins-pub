# ✅ Ready to Push to GitHub!

Your code is ready. Here's what to do:

## 🚀 Quick Steps:

### 1. Create GitHub Repository

1. Go to: **https://github.com/new**
2. Repository name: `goggins-pub`
3. Description: "Goggins of Monkstown website"
4. Choose **Public** or **Private**
5. **IMPORTANT:** Do NOT check "Add a README file" or any other options
6. Click **Create repository**

### 2. Push Your Code

After creating the repo, GitHub will show you commands. Copy and run these in your terminal:

**Replace `YOUR_USERNAME` with your actual GitHub username:**

```bash
cd /Users/markgriffith/goggins-pub

git remote add origin https://github.com/YOUR_USERNAME/goggins-pub.git

git branch -M main

git push -u origin main
```

You'll be asked for your GitHub username and password (or token).

### 3. Import to Vercel

1. Go back to: **https://vercel.com/new**
2. Click **Import Git Repository**
3. Your `goggins-pub` repo should now appear!
4. Click **Import** next to it
5. Vercel will auto-detect Next.js
6. Click **Deploy**

### 4. Add Your Domain

After deployment completes:

1. In Vercel dashboard, click your project
2. Go to **Settings** → **Domains**
3. Add: `gogginspub.com`
4. Add: `www.gogginspub.com`
5. Follow Vercel's DNS instructions

### 5. Update DNS

At your domain registrar (where you bought gogginspub.com):
- Add the DNS records Vercel provides
- Wait 5-60 minutes for DNS to propagate
